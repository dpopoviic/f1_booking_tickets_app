Formula 1 Ticket Booking Application
A ticket booking system for Formula 1 races built with C# .NET backend and React frontend.
-Project Overview
This application allows users to purchase tickets for Formula 1 races, manage their bookings, and provides organizers with real-time sales analytics through a dedicated reporting portal.

Key Features
Ticket Purchase - Book tickets for single or multiple race days
Flexible Seating - Choose from different seating zones with dynamic pricing
Early Bird Discounts - Automatic 10% discount for early bookings
Promotional Codes - Support for discount codes and referral bonuses
Multi-Currency - Purchase tickets in different currencies with live exchange rates
Ticket Management - Modify or cancel existing bookings
Admin Dashboard - Organizers can view real-time sales statistics
